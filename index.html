Patience is a Virtue

I have spent most of the afternoon plugging away at this. Downloaded and installed Git (version control software). Created my first repository called hello-world. Launched Git BASH, which I think is like a command shell, and I created a Hello-World directory and README file.

It took me three or four attempts before I was able to commit the README file and another two before it's contents displayed properly. Next I created this page, checked out the files, modified them locally and commited them back to GitHub. Woo hoo!

Remember these Git BASH commands

mkdir ~/Hello-World

cd ~/Hello-World

git init

touch README

git add README

git commit -m 'first commit'

git remote add origin https://github.com/franpea/Hello-World.git

git push origin master

git fetch origin

git checkout gh-pages

git add index.html
